# Localisation Workbench

A Python-based toolkit for converting and comparing localisation files across CSV, JSON, and Excel formats.

## Current Features

- Convert CSV to JSON
- Convert Excel to JSON
- Convert JSON to Excel
- Compare JSON files and generate a report

## Project Goal

This project is being developed into a lightweight product demo to show:
- Python development
- file transformation workflows
- data validation thinking
- technical product skills

## Current Files

- `Multicolumn_excel2json.py`
- `JSONcompare.py`
- `JSON_converter.py`
- `JSON2Excel.py`
